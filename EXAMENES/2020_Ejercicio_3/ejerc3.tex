Se considera una viga en ménsula con las dimensiones y geometría indicadas
en la figura. En el extremo libre actúa una carga vertical descendente
$P=10$ kN. El módulo elástico es $E=20$ GPa y el coeficiente de Poison
$\nu=0$. Se considerará la hipótesis correspondiente a tensión plana, con
un espesor $t=1$ m.

Para resolver el correspondiente modelo de elementos finitos, con el
programa {\tt FEAP},  se considerará
una malla con $11 \times 3$ {\em nodos}, discretizada con elementos
cuadriláteros de $4$ y $8$ nodos, y elementos triangulares de $3$ y $6$ nodos.

{\bf NOTAS:}
\begin{enumerate}

\item La malla se generará con la instrucción {\tt block}, tomando las
esquinas en el orden indicado en la siguiente figura.
\item Los elementos triangulares tendrán la hipotenusa en la dirección
$1-3$ de las esquinas del bloque.
\item La fórmula de Resistencia de Materiales que permite calcular la
flecha en el extremo es: $f=\frac{1}{3} \frac{PL^3}{EI}$.
\item {\em Se cargará el fichero postscipt de contornos de movimientos en dirección vertical para la malla de elementos cuadriláteros de 4 nodos}
\end{enumerate}
